# 小S学语文

## 题目描述

“得语文者得天下！”小S深刻铭记着自己的使命。为了更好地备战期中考试，小S决定写一些作文来练练手。

当然，这一切都被张老板监视着。原本小S写好交给夫子的作文全都通过某种神秘力量落入了张老板手中。

由于张老板沉迷凹深渊，忘记了作文分数的正常取值范围，所以他给小S的作文批改的分数为[1,1000]内的任意一个整数。

而当小S看到自己跌宕起伏的分数时，凭借自己丰富的经验，他在电光火石之间想到了张老板。

“张-希-娜！把你世界的资源全部给我！”小S带着手下的内卷悍将怒气冲冲地找到了正在等树脂刷新的张老板。

“先等等，你看我给你改的分数起伏很大，你不就可以选出连续的若干篇作文，使得作文的分数单调递增。你不就可以用最高分减去最低分代表你进步的分数，去小Y那里炫耀了嘛。”

小S恍然大悟，旋即按顺序拿出了自己写过的作文。但当他仔细一想，旋即又发现了不对劲的地方：

“不过，为什么一定要连续的区间，还要单调递增呢？”（可参照样例解释）

“这个嘛，你问小Y喽！”说完，张老板头也不会地刷圣遗物去了........

不过由于小S还要偷偷准备满分作文，在考场上惊艳所有人。于是就将求出最大进步分数的任务交给了无私的你。

## 输入格式

一行，若干个整数，按顺序给出小S每一篇作文的得分。

## 输出格式

一个整数，表示小S最大的进步分数。

## 样例输入

```
1 1 4 5 1 4

```

## 样例输出

```
4

```

## 提示

满足索引连续且分数单调递增的区间有[1,4],[1,4,5],[4,5],[1,4]。其中进步分数最大的区间为[1,4,5],进步分数为 4。

由于小S还要复习其他科目，所以保证给出的作文数量不超过1000。
