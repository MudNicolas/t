# 小S的考试

## 题目描述

“内卷才是第一生产力！！！”怀着拿下满分的信念，小S步入了化学考试的考场。

化学题库中共有 n 道题目，每道题目由题面和答案组成，都是一个字符串。为了增大小S的贺题难度，保证所有题目题面互不相同。

本次考试包含 q 道题目，均在题库中出现过。每道题目都有 4 个选项，小S需要从 4 个选项中选出与答案相符的选项。

“这次我一定能发挥出自己的真实水平，再也不让张老板有机会嘲讽我！！！！”

心想着，小S默默打开了自己随身携带的小抄本，翻开了化学题库......

然而，小S此时还处于自闭状态中，无法正确配对考试的题目和题库中的题目，于是就把这个光荣的任务交给了你。

## 输入格式

第一行两个正整数 n,q。

接下来 n 行，每行 2 个用空格分隔的字符串，表示这道题目的题面和答案。

接下来 q 行，每行 5 个用空格分隔的字符串，第一个字符串表示模拟考试中这道题目的题面，其余 4 个字符串按顺序分别为这道题目的选项 A 到选项 D，保证选项各不相同。

## 输出格式

对于考试中的每道题目，输出一个字符表示这道题目答案对应的选项，保证所有题目均有解。

## 样例

### 样例输入

```
3 4
decoak yes
duliuchutiren nonono
csps noiptg
decoak yes no qwq qaq
csps noiptg noippj noi cspj
decoak qwq qaq yesyes yes
duliuchutiren yes no nono nonono
```

### 样例输出

```
A
A
D
D
```

## 提示

令 s 为输入中字符串长度的最大值。保证 n,q,s<=100，所有字符串都由小写字母组成。
