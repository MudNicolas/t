# 小S学数论

## 题目描述

作为地基一班的内卷委员会主席，小S在开学的半个月内就刷完了五本数学教辅。并且励志成为一名数学大师！

这让一直在肝原神的张老板感到不安，于是他偷偷把自己的“数论书”塞给小S，并且在他耳边悄悄说道：

“这本书上的内容你一点都不会，所以你还不能算一个合格的数学大师！要想成为一名真正的数学大师，你必须学会这些筛法”

小S默默地接过了张老板的书并迫不及待地翻看起来，只不过在他脸上写满了疑惑：

“线性筛（xxs）、杜教筛、洲阁筛、Min_25筛......这些都是什么，我怎么一点都看不懂....”

张老板轻轻拍打了小S的肩膀，悠哉地说道：

“这些内容对于你们这些初学者来说确实不友好了，不过我相信最简单的埃拉托色尼筛法我相信你还是可以学会的。”

小S将书本还给张老板，拿起了他写的密密麻麻的错题本，摇了摇头说道：

“学这些有什么用，能够让我干碎【】班吗？还不如多卷几道圆锥曲线！”

不料张老板对此早有预测，于是他拿出了自己的杀手锏：

“要是你学会了埃拉托色尼筛法，小Y可是会很崇拜你的哦......”

此刻，小S以常人难以想象的速度从张老板手里抢回“数论书”，穿云裂石地喊道：

“快教我！！！！！”

--------------------------分割线-------------------------

埃拉托色尼筛法是一种著名的素数筛法，可以查找所有直至 n 的素数。该算法的步骤是：

1. 写下 2 到 n 之间的所有整数（包括 2 和 n）。

2. 找到尚未删除的最小数，并将其命名为 p； 则 p 是素数。

3. 划掉 p 所有尚未划掉的倍数（不包括 p）。

4. 如果尚有数未被划掉，请转到步骤 2。

张老板给出了n，由于他今天的委托还没做，于是又给出了一个k。而小S的任务就是找出第k个被划掉的数。

不过，此时的小S脑海中全是如何向小Y炫耀的画面，于是只好向你求助。

## 输入格式

一行两个整数n和k，具体含义请见题目描述。

## 输出格式

一行一个整数，表示第k个被划掉的整数。

## 样例输入

```
15 8
```

## 样例输出

```
15
```

## 提示

第一次筛数 p=2，划去 [4,6,8,10,12,14]；

第二次筛数 p=3，划去 [9,15]，此时15恰好是第8个被划掉的数。

对于所有数据，保证 2<=k<n<=1000。

在你的帮助下，小S以第三宇宙速度跑到小Y面前，展示了自己“新学的强大筛法”。

“您的话真怪。”小Y鄙夷不屑地――他有这样的感觉――微笑着说。

小S再也说不下去,就离开班级,尽力忍住翻腾着的泪水,不让自己哭出声来。


此时，躲在暗处的张老板偷偷地笑了..........
