# 优雅数

## 题目背景

给定两个数 L 和 R，求 L 到 R 之间（包括 L 和 R )有多少个数是“优雅”的。
对于一个 n 位的优雅数（没有前导零）， n 个数字中 n-1 个数字相同，有且仅有一个数字不同。例如，33323、110 都是优雅的，而 9779、55555 都不是优雅的。

## 输入格式

一行两个数 L 和 R。

## 输出格式

一行一个数，表示 [L,R] 区间有多少个数是优雅的。

## 样例输入

110 133

## 样例输出

13

## 提示

在 [110,133] 的范围内，110 112 113 114 115 116 117 118 119 121 122 131 133 是优雅的。
对于 100%的数据满足：100≤L≤R≤1000
