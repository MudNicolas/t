# Nicolas摘苹果

## 题目描述

Nicolas 家的院子里有一棵苹果树，每到秋天树上就会结出 $10$ 个苹果。苹果成熟的时候， Nicolas 就会跑去摘苹果。 Nicolas 有个 $30$ 厘米高的板凳，当他不能直接用手摘到苹果的时候，就会踩到板凳上再试试。

现在已知 $10$ 个苹果到地面的高度，以及 Nicolas 把手伸直的时候能够达到的最大高度，请帮 Nicolas 算一下他能够摘到的苹果的数目。假设他碰到苹果，苹果就会掉下来。

## 输入格式

输入包括两行数据。第一行包含 $10$ 个 的整数（以厘米为单位）分别表示 $10$ 个苹果到地面的高度，第二行只包括一个的整数（以厘米为单位），表示 Nicolas 把手伸直的时候能够达到的最大高度。

## 输出格式

输出包括一行，这一行只包含一个整数，表示Nicolas能够摘到的苹果的数目。

## 样例 #1

### 样例输入 #1

```
100 200 150 140 129 134 167 198 200 111
110
```

### 样例输出 #1

```
5
```
