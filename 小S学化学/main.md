# 小S学化学

## 题目描述

作为取得多项优异成绩的前2024届化竞团队的一员，化学课本上的知识对小S来说几乎是有手就行。

M老师在讲到原子结构这一章的时候，小S认为这部分内容实在太简单，于是将这几节课的宝贵时间抽出来给小Y写道歉信。

由于这几天的内卷任务比较重，小S每天仅能抽出114514ms来写化学作业。

而这样导致的后果就是，小S的化学作业本上出现了密密麻麻的红叉，并且被M老师在课堂上公开处刑。

为了检验小S是否已经完全掌握原子结构的知识，M老师给出了一个n，代表一种元素的原子序数，小S的任务是在黑板上写出该元素的核外电子轨道排布式。

“如果有一点点错误，我让你把化学书抄五百遍！”M老师毫不留情地对小S说道。

而此时小S还需要修改写给小Y的道歉信，他只能卑微地向你求助。

（ps：元素核外电子轨道排布式可参照化学选修二的教材）

## 输入格式

一个整数n，表示该元素的原子序数。

## 输出格式

若干行，表示该元素的核外电子轨道排布式（可参照样例输出格式）

## 样例输入

```
56
```

## 样例输出

```
1s2
2s2 2p6
3s2 3p6 3d10
4s2 4p6 4d10
5s2 5p6
6s2
```

## 提示

n<=118，保证所有给出的元素核外电子填入能级的顺序均符合选修二课本上给出的构造原理，且最多只发生一次相邻填入的两个能级之间电子的跃迁。

（如Cr、Cu，数据中不包含Nb、Pd、Au等排布较为复杂的元素）

在你强势的帮助下，小S成功通过了这次魔鬼考核。一下课，他就迫不及待地将道歉信递给小Y。

不幸的是，小S并没有得到小Y任何的回复。只是在几天以后，小S在某人口中听到了这样的话：

“1500字，你们学竞赛的都这么闲的吗......”

此时，躲在暗处的张老板偷偷地笑了..........
