# G老师的量子力学

## 题目描述

G老师曾经说过:"Ag离子不带电，这一点学过化学的人都知道！"

最近G老师沉迷于量子力学无法自拔，整天都在办公室思考这个深刻的问题：量子的叠加态和纠缠态是否会推翻现有的哲学理论。

在G老师自学了量子力学的基础之后，他以超强的天赋独自证明和推演了贝尔不等式。现在他想考考你对量子力学的理解。

G老师为了表达方便，他把量子元素简化成了A-Z 的26个大写字母，他会给你一个包含n个量子元素的量子序列a。

G老师定义一个量子序列是否纠缠态为:你可以任意排列这个量子序列。在所有排列中，如果存在一种排列方式，满足任意 0<=i<=n-3,a[i]<a[i+2] ，则这个序列为纠缠态的量子序列。

你的任务是，判断序列a是否为纠缠态的量子序列。如果是则输出Yes,否则输出No。

## 输入格式

由大写英文字母构成的序列

## 输出格式

如果是纠缠态的量子序列输出"Yes",否则输出"No"。

## 样例

### 样例输入

XNZHANINAGR

### 样例输出

No

## 提示

len(a)<=100

注意可以任意排序，以及 a[i]<a[i+2]
