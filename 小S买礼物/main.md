# 小S买礼物

## 题目描述

临近小Y的生日，小S暂时放下了手中的内卷工具前往礼物店给小Y挑选礼物。

由于小S怀疑有人跟踪，于是他排在了队伍的末尾，仔细观察前面的每一个人。

经过小S细心的观察，他发现有些人在队伍中只是在等待他的朋友而并非在排队买礼物。

经过强大的化学推导，小S发现站在一个人后面的他的朋友都穿着和他相同颜色的衬衫。

就在此时，小S注意到他前方一个熟悉的.....蓝色封面的书。

“糟糕，我五三忘记做了！”

小S突然想起今天的内卷任务还没有完成，于是他想要知道他现在实际排在第几位，便于他计算什么时候才可以开始内卷。

由于小S还要花费大量时间计算小Y喜欢什么礼物，于是就把这个艰巨的任务交给了你。

## 输入格式

一行，若干个由空格隔开的字符串，表示小S前面的人的衬衫颜色。所有字符串的大小写严格区分。

## 输出格式

一行，表示小S当前实际排在队伍的第几位。

## 样例 #1

### 样例输入 #1

```
D D d P X X CA CA Ca
```

### 样例输出 #1

```
7
```

## 提示

D D 实际是一个人在排队， d 是一个人在排队，P 是一个人在排队， X X 实际是一个人在排队， CA CA 实际是一个人在排队，Ca是一个人在排队，所以小S实际排在第 7 位。

两个相邻的群体、相邻的个人或相邻的个人和一个集体将永远不会穿着相同颜色的衬衫。

“呼，终于可以继续卷五三了！”

小S激动地打开他的五三，只发现上面打满了五颜六色的√。

“不会真有人跟踪我吧.......”
