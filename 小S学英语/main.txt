# 小S学英语

## 题目描述

作为地基一班内卷事业的忠实崇尚者、自觉遵守者、坚定捍卫者，小S的目标是努力使尊卷学卷守卷用卷在全班级蔚然成风。

“天道酬勤！”小S将英语老师Randy的话铭记于心。

张老板凭借着地基一班前英语课代表的身份，默默地送给了小S一份英语试卷大礼包。并且庄重地对小S说道：

“现在，我代表Randy宣布两件事：第一件，你要在1.14514s内写完这份试卷；第二件，Randy这个星期不会再布置任何作业。”

“什么，为什么我不能卷更多作业！！！”小S目瞪口呆地说道。

“好的，既然你对第一件事情没有意见。那么先等你写完这份试卷，我们再讨论第二件事情吧。”张老板早有预料地说道。

小S只好默默地翻开了英语试卷。而当他勉强写出七选五中的两道题时，却发现时间已经来不及了。

而小S为了维护自己的尊严，剩下的三道题他决定随机不重复地等概率选取三个答案。

（例如小S目前的作答情况为A_B__，则小S可能的最终答案中包含 ACBED、AEBDG、AFBEG等情况）

张老板凭借着常年漫游提瓦特大陆积累的丰富经验，一眼就看出了小S的诡计，于是他毫不留情地对小S说道：

“如果你能算出你这样做的期望得分，那么我就算你七选五满分。不过如果你算错了，那么小Y会很鄙视你的。”

当然，小S早已忘记期望的定义，于是张老板贴心地给出了一点小提示：

“假如你当前的答案是ABCD_，标准答案为ABEDF，那么你最终答案的情况就有 ABCDE，ABCDF，ABCDG，得分分别为6,8,6。

而你的期望分数也可以理解为你所有情况下的平均分数，那么你在这种情况下的期望得分就是(6+8+6)/3≈6.67”

然而，当随机填入的答案数量增长到3个时，小S感到亿丝丝绝望。

不过小S为了维持自己在小Y心目中的形象，还是决定向你求助。

## 输入格式

第一行输入一个长度为2的字符串，表示小S前两题的作答情况，保证其组成为两个不同的大写字母

第二行输入一个长度为5的字符串，表示标准答案，保证其组成为五个不同的大写字母。

## 输出格式

一行一个实数，表示小S的期望得分，四舍五入保留两位小数。

## 样例输入

```
AB
ABCDE
```

## 样例输出

```
5.2
```

## 提示

在你的帮助下，小S毫无疑问地回答出了正确答案。

当他自豪地向小Y提起这件事情时，只得到了小Y冷冷的回复：

“。。。你戏真多。。。”

小S的笑容消失了，眉头痛苦地皱起来。

此时，躲在暗处的张老板偷偷地笑了..........
