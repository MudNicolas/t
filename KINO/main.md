题目描述

电影院的每排有 N 个座位。座位分为普通座位 S 和爱心座位 L。其中，爱心座位总是两两出现。

每排座位的相邻两个座位之间都设有一个杯架（除了两个爱心座位之间外）。特别地，一排座位的两侧也都设有杯架。

现有 N 位顾客坐满这一排的 N 个座位，求最多有多少人能够使用杯座。

输入格式

一行，N 个字符 S/L 构成的字符串，分别表示普通座位和爱心座位。

输出格式

输出能够使用杯座人数的最大值。

样例输入 #1

SSS

样例输出 #1

3

样例输入 #2

SLLS

样例输出 #2

4

样例输入 #3

SLLLLSSLL

样例输出 #3

7

提示

【样例 3 解释】

若用 * 来表示杯座，那么座位情况如下：

* S *L L* L L *S* S *L L*

【数据规模与约定】

1 <= N <= 50。
