题面：
    由于 st210800020481 风见喵喵子同学的迷迭香经过了 710 天的等待后终于有了一套新时装，st210800020481 风见喵喵子同学很开心，为了纪念这一伟大的香厨大胜利，她特地找到了同为香厨的高二(8)班的 CH 同学来帮忙出一道题。
    这道题是这样的。在一个平面直角坐标系中，分布着 n 个点，这 n 个点每个点都有一个坐标 (xi , yi) ，现在风见喵喵子同学想让你告诉她，这 n 个点能构成几个正方形。因为风见喵喵子很可爱，所以你要帮帮她。但 CH 同学认为这题太简单了，所以点的个数 n 被扩大到了 1000。但是可爱的风见喵喵子同学不想让这题太难，于是所有的点都在第一象限内且 0 <= xi , yi∈整数 <= 100。所以请你帮助风见喵喵子做出这道题。
    简单来说，给你 n 个点，你需要求出这 n 个点能构成的正方形个数。

输入格式：
    第一行一个 n 表示点的个数。
    接下来 n 行，每行两个数，表示这个点的坐标。

输出格式：
    仅一个数，能构成正方形的个数。

Input01 ：

7
1 0
0 1
1 2
3 3
2 1
4 0
5 2

Output01 ：

2

提示：
    n <= 1000 , 0 <= xi , yi <= 100
    可以考虑一下正方形的判定定理，然后思考一下 0 <= xi , yi∈整数 <= 100 是为什么。
