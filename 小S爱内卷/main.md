# 小S的最大子段和

## 题目描述

Nicolas 给小S出了n道高数题，每道题有一个价值 $a_i$，表示这道题对小S内卷能力的提升程度。由于 Nicolas 是反内卷斗士，所以 $a_i$ 可能为负数。

由于 Nicolas 喜欢求导，所以他要求小S做的高数题必须是一段连续的区间。

小S同学忙于管理班级事务，所以请你帮他选出一段连续且非空的区间，使得这段区间对小S内卷能力提升最大。

## 输入格式

第一行给出若干个整数，第 i 个整数表示第 i 道高数题对小S 内卷能力的提升程度为 ai。

## 输出格式

输出一行一个整数表示答案。

## 样例

### 样例输入

2 -4 3 -1 2 -4 3

### 样例输出

4

## 提示

选取高数题 [3,5] 子段 {3, -1, 2}，其和为 4。

可使用split()以空格分割字符串
若 s="12 54 64" 的字符串，可使用 list(map(int,s.split())) 将s转换为整数列表[12,54,64]

保证 n<=10000，-10000 <= ai <= 10000。
