# 小S转进制

## 题目描述

今天小S在信息课上学会了二进制转十进制，比如10101B ，那么它的十进制表示的式子就是 :

$1*2^4+0*2^3+1*2^2+0*2^1+1*2^0$

小S非常激动，于是赶紧跑到小Y面前炫耀。然而，小Y因为一些离谱的事情正在自闭中,于是他十分无情地泼了小S一盆冷水：

“换成一个特别大的M进制数N，把它转为N，你还会吗？”

由于小S今天内卷消耗了太多能量，此时脑子一片空白，于是得到了小Y无情的嘲讽。

小S环顾四周，只好向你求助。

注意：当系数为0时，省略该项。

## 输入格式

两个数，M和N，中间用空格隔开。

## 输出格式

共一行，一个十进制表示的式子。

## 样例 #1

### 样例输入 #1

```
2 10101
```

### 样例输出 #1

```
1*2^4+1*2^2+1*2^0
```

## 提示

对于100%的数据，1<M<10，N的位数不超过1000。

小S非常急切地希望你做出这道题，来挽回他在小Y面前的形象。
