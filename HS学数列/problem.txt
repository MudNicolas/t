HS学数列

题目描述

High_Score开始学数列了，但是Ta遇到了一个问题:Ta缺少刷题量，做题很慢。

于是High_Score找到了万能的H老师，由于H老师正在爆切lOl，没空给High_Score讲题，所以他给了High_Score一道题:

已知长度为n的数列 {b_n} ，求数列 {a_n}

数列 {b_n} , {a_n} 满足一下关系:

b_i=(a_1+a_2+…+a_i)/i

Input

一行 n 个整数，表示数列 bn 。

Output

一行 n 个整数，表示数列 an 。

Example1

Input
2

Output
2

Example2

Input
3 2 3 5

Output
3 1 5 11

Example3

Input
1 2 2 3 4

Output
1 3 2 6 8

Note
对于100%的数据,保证 1 <= n <= 1e4 , 1 <= b_i <= 1e9
