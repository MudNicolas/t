# 小S学技术

## 题目描述

作为地基班公认的“技术之神”，小S不仅擅长Ai、Fw、Fl、Br、Ae、Pr、Id、Ps等软件的安装与卸载，

精通CSS、Javascript、PHP、ASP、C、C＋＋、C#、Java、Ruby、Perl、Lisp、python、Objective-C、Actionscript、Pascal等单词的拼写，

熟悉Windows、Linux、Mac、Android、IOS、WP8等系统的开关机；

同时熟练掌握淬火、回火、正火、退火等各项技能，独自开发了三视图补线的“11个秒杀定定理”以及设计题的“17个必备模型”。

这一天，原本应该上技术课的小S受邀来到了张老板的尘歌壶。同时被邀请的还有Mandela、Teacher_Ma，Dumby_Cat，LH-2，Beauty等一众技术神仙。

“由于wheneveright和_WD_临时参加牛X网的比赛无法到来，我们就开始吧。”张老板微笑着说道，同时命令手下的旺财给所有的技术之神递上了一份地图。

“今天我把各位技术之神邀请过来，主要是想玩一个游戏。当然，鉴于各位强大的技术能力，主要还是推演.....”张老板说着，同时缓缓站起身来。

“你个**********，有什么话快说，不要影响我内卷！”小S发觉他已经有114514ms没有写代码了，感到十分愤怒。

“当然，这次游戏的结果会公布出来，所有我们认识的人都会看到哦...”张老板慢悠悠地说道，意味深长地看了满脸通红的小S一眼。

“既然大家都没有意见。那么我就简单介绍一下这次的游戏规则吧。”说着，张老板露出了慈父般的笑容。

【题目描述】

提瓦特大陆上发生了一场战争。公子用愚人众复制后数量庞大的百无禁忌里残存的岩神神力和冰皇赐予自己的执行官力量，来暂时解封被岩枪镇压在海里的远古魔神奥塞尔，逼迫摩拉克斯现身。

仙人与七星本正在璃月港口对峙，值此山雨欲来之际，两方暂时放下不和，共同迎击远古魔神。

漩涡之魔神奥赛尔有n干个弱点，他的每个弱点都具有一定的防御力。

群玉阁上有m个归终机，每个归终机都有一定的攻击力；

当这个归终机的攻击力大于弱点的防御力时，那个弱点就会被破坏掉。否则弱点不受影响。

当奥赛尔的弱点全部被破坏掉之后，所有的攻击都会攻击到奥赛尔本身上去造成伤害。

（即每个归终机要么破坏一个弱点，要么攻击奥赛尔本身）

在瞬息万变的战场中，选择一个最优的攻击对象是非常重要的。

因此，张老板想让技术之神写出一个战斗系统出来，判断出群玉阁最多能对奥赛尔造成多少伤害。

当然，小S由于消耗了大量的能量与张老板对线，所以只能向你求助。

## 输入格式

第一行n个整数，表示每个弱点的防御力。

第二行m个整数，表示每个归终机的攻击力。

## 输出格式

一个整数，表示对奥赛尔造成的最大伤害值。

## 样例输入

```
1000 2000 1200
2100 2000 1200 1000 1000
```

## 样例输出

```
2000
```

## 提示

n<=1000，m<=1000

奥赛尔有3个弱点，防御值为1000(a),2000(b),1200(c)；

群玉阁上有5个归终机，攻击值为2100(d)，2000(e),1200(f),1000(g),1000(h)。

攻击的最优方案是d攻击b，e攻击c，f攻击a，g和h攻击对方母舰本身，造成2000点伤害。

（来自希娜小姐的小提示：很显然要用最低的攻击值之和去破坏所有的弱点。对于防御力从小到大的每个弱点，用最小的攻击力大于该防御值的归终机去破坏即可。

那么剩下的归终机都可以直接对奥赛尔本身造成伤害，同时还要注意无论如何都无法破坏所有弱点的情况）

在一众技术之神的合作下，小S很轻松地完成了这个游戏。

就当小S想要让张老板夸赞自己时，他突然发现，自己的名字出现在了大屏幕上：

“归终机Mandela发起了进攻，小S成为一个被击败的弱点！”

小S默默地望向璃月港，他的笑容消失了，眉头痛苦地皱起来。

“真怪!”张老板又说了一遍，接着摇摇头，向绝云间那边走去。
