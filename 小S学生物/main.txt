# 小S学生物

## 题目描述

为了营造良好的内卷氛围，小S不仅推广并严格执行“入室即静，入座即学”的八字方针，同时常常利用闲暇时间巡视各位卷王的内卷状况。

这天，小S无意中发现还有20页生物作业本、400道判断题要写的生物竞赛卷王Mulberry在下课时放下了笔，拿起了零食。这令小S感到十分不爽。

“你个卷柏，《教材帮》都没卷完就和朵拉在教室里瓜分食物？”小S怒气冲冲地翻开了Mulberry的作业本，指着上面大片的空白说道。

“哎呀，卷柏不是刚从杭州卷完竞赛回来，他想休息一下不行吗？”小S顺着声音的方向望去，只见张老板抱着小旺财春光满面地走来。

然而，本着“只要卷不死，就往死里卷”的基本原则，小S此时听不进任何其他劝阻的话。不过由于Mulberry的强大内卷实力，生物学考拿了S+，所以小S的态度还算温和。

“要不这样吧，小S你先做一道生物题试试吧，我觉得你会理解卷柏的。”张老板默默地说道，不过显然这道题将由他“精心准备”。

“你要让我少背5个英语单词来做一道生物题？开玩笑，我的内卷时间可是很珍贵的！”小S不屑地说道。不过凭借着以往的惊艳，他似乎知道张老板接下来要说什么...

“小S，你也不想让别人知道你强大的控分能力吧......”小S默默地翻开了某一次期中考试的排名，他看到了他自己的名字，以及在他名字上面另一个很熟悉的名字。。。

“来吧！”小S恍然大悟，同时咬紧牙根，他心里暗下决心再也不控分了。

【题目描述】

张老板在金苹果群岛新投资了一家生物实验室，主要研究蟋蟀的遗传。

根据刻在DNA里的9:3:3:1，张老板很快通过求导得出了他培育的蟋蟀一共有n种不同的基因型。

然而，不幸的是，由于“看不见的手”，如果出现了两只基因型相同的蟋蟀，那么张老板能够抽卡的原石就会大大减少，这显然是他不能接受的。

于是，他想要知道，当每蟋蟀的基因在n种基因型中等概率形成时（即每只蟋蟀为任意一种基因型的概率都是1/n）。至少有多少只蟋蟀，使得出现任意两只相同基因型的蟋蟀的概率会大于等于k%。

显然，这个综合性的问题被交给了小S。不过他还在思考他是如何做到精准控分的，所以这个问题只好交给你来解决。

## 输入格式

两个整数n,k，表示基因型的数量，张老板要求出现两只相同基因型的概率百分数。

## 输出格式

一个整数，表示最少的蟋蟀数量。

## 样例输入

```
365 50
```

## 样例输出

```
23

```

## 提示

1<=n<=10000,0<=k<=100。

样例也可以这样理解：每个人的生日在一年365天中等概率出现，使得有任意两个人生日相同的概率大于50%的最少人数为23。

当小S回过神来以后，发现Mulberry早已瓜分完了他所有的零食。张老板和他的小旺财也不见任何踪影。

“怎么分零食都不带上我的......”小S望着空空如也的教室，默默地打开了他的书包，“帝君，带我回璃月.......”
