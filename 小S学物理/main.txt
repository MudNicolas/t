# 小S学物理

## 题目描述

“没有作图就没有物理解题！”――――物竞王牌教练J。

身为担当民族复兴大任的时代新人，小S不仅致力于研究 6nm 的芯片，而且对各类物理问题十分感兴趣。

面对即将到来的物理学考，这天，身为二中彭于晏的小洋找到了小S。就当他准备向小S求助的时候，张老板通过传送锚点偷偷来到了小S的身后。

早已习惯了张老板各种搞怪操作的小S对这位不速之客的到来反应十分平淡，他冷冷地说道：

“希-娜-小-姐，请问你不去肝沙漠书，在这里干什么？”

然而，张老板只是微微冲小S和小洋笑着，并没有说任何话。

短暂的沉默过后，小洋首先开口说道：“尊敬的小S，你的物理复习卷能不能给我做个参考系。”

显然，沉迷于研究苹果芯片的小S只在试卷上写下了自己的大名。然而他并不想在帅哥面前丢脸，于是他拍着胸脯说道：

“给我1h22min，我速通复习卷！”

此时，一直沉默不语的张老板悄悄凑到小S耳边，用只有他能听到的声音说道：

“别忘了，他和小Y可是一个班的哦......你如果速度太慢......”

没等张老板说完，小S拦住刚转身准备离开的小洋，用坚定的眼神说道：

“1min22s！你在这里等着，我现在去拿试卷！”

“哎哎哎，你想速通？先把这道题做了吧。”张老板拦住小S，并且指了指不远处的史莱姆。

【题目描述】

张老板在提瓦特大陆上漫游时，遇到了一只奇怪的史莱姆。具体地，他是一个二维生物，并且在直角坐标系上的坐标为A（x0，y0）。

由于某种来自张老板的钞能力，史莱姆沿x轴负方向以某一速度抛出，且只受到竖直向下的重力作用做平抛运动（g取10m/s），最后恰好以速度v落到原点（0,0）。

张老板通过强大的高数分析，得到了史莱姆落到原点时的速度v，以及v与竖直方向的夹角θ。

为了快速获得史莱姆凝胶，张老板想让身为逆向人的小S计算出史莱姆的初始坐标A（x0,y0）。

然而，小S一心只想速通复习卷，于是把这个简单的任务抛给了你。

## 输入格式

两个实数，表示史莱姆落到原点时的速度v，v与竖直方向的夹角θ。

## 输出格式

两个实数x0，y0，表示史莱姆的初始坐标，四舍五入保留两位小数。

## 样例输入

```
14.142136 0.785398
```

## 样例输出

```
10.00 5.00

```

## 提示

1<=v<=100,π/12<=θ<=5π/12。

可以使用 math 库里的 math.sin() math.cos() 进行计算。

“呼，终于做完了！”就当小S兴致勃勃地想要把卷子交给小洋时，却发现走廊上只剩下正在肝沙漠书的张老板。

“他........去哪了？”小S满脸困惑地说道。

“哦，我1min21s前就把试卷给小Y，现在他们班的人都知道答案....所以小洋也就走了。”张老板默默地听着派蒙介绍任务，回答道。

“你-干-嘛-哇！！！！”小S默默地戴上了痛苦面具，呆呆地望着远处的摘星崖沉默不语.......
