金友滨老师正在讲二维数组，他现在想对一个二位矩阵进行一些操作，这是同学们的最终目标。

1 操作，进行水平翻转（即水平对称）

2 操作，进行竖直翻转（即竖直对称）

现在想让你计算操作完成后的矩阵是什么样的。

第一行为 n ，表示这是一个 n*n 的矩阵。

接下来 n 行，每行 n 个数，描述这个矩阵。

接下来一个字符串，只会出现 1,2 中的一个操作。

Input#01 :

3
0 1 2
2 1 0
1 0 2
212

Output#01 :

2 1 0
0 1 2
2 0 1

解释：

1 0 2    2 0 1    2 1 0
2 1 0    0 1 2    0 1 2
0 1 2    2 1 0    2 0 1

n <= 10 ，操作数 <= 15
