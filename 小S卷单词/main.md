# 小S卷单词

## 题目描述

众所周知，小S领导着地基班的伟大内卷事业。Nicolas为了检验小S的内卷实力，从数据库里随机挑选了N个单词给小S，由于数据库遭到反内卷协会的袭击，单词可能重复出现。

Nicolas给小S的任务是，找出这N个单词中有多少个不重复的单词。

由于小S同时还要准备文艺汇演，所以就将这个光荣的内卷任务交给了你。

## 输入格式

一行，以空格分隔的所有字符串

## 输出格式

一行，表示共有几个不重复的字符串

## 样例 #1

### 样例输入 #1

```text
xxs suck xxs
```

### 样例输出 #1

```text
2
```

可以参考python语法手册中Set的用法
